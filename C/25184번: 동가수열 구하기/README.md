## 문제

[문제 링크](https://boj.kr/25184)

<p>길이가 $N$인 동가수열은 다음 두 조건을 만족하는 수열이다.</p>

<ul>
<li>동가수열은&nbsp;$1$ 이상 $N$ 이하인 정수로 이루어져 있고, 모든 원소는&nbsp;서로 다르다.</li>
<li>동가수열의 서로 이웃한 원소의 차는 $\lfloor \frac{N}{2} \rfloor$이상이다.</li>
</ul>

<p>길이가 $N$인 동가수열을 아무거나 하나 구해보자. 주어지는 모든 입력에 대해 동가수열은&nbsp;항상 존재한다.</p>

## 입력

<p>첫째 줄에 구하고자 하는 동가수열의 길이 $N$($1 \le N \le 5\,000$)이 주어진다.</p>

## 출력

<p>첫째&nbsp;줄에 길이가 $N$인 동가수열을 아무거나 하나 출력한다.&nbsp;동가수열은 공백으로 구분해서 출력해야 한다.</p>
